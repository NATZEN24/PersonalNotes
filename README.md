# Notes App

A simple notes application built with HTML, CSS, and JavaScript.

## Features
- Add notes
- Delete notes
- Notes are saved in localStorage
- Data persists after page refresh

## Technologies
- HTML
- CSS
- JavaScript 

## Purpose
This project was created to practice basic CRUD operations and DOM manipulation.

## How to use
1. Enter a note in the input field
2. Click "Add"
3. Notes are saved automatically
